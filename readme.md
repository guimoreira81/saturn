# Saturn
A personal project that tests 3d rendering using only characteres.

**`game.js`**: Javascript code that renders 3d objects using colored characters.
**`script.js`** Javascript code that creates Saturn and it's rings.

## Controls
- `w`, `s`, `a`, `d` for moving;
- `z` and `x` for moving up and down;
- `q` and `e` for rotating;
- `arrow keys` for moving the camera.
